# HR-Attrition-Dashboard-Advanced-Excel

Project Overview

This project analyzes employee attrition data using Microsoft Excel to identify workforce trends, attrition patterns, and key HR metrics. The dashboard provides insights into employee turnover, job satisfaction, and demographic distribution to support better decision-making.
________________________________________
Objectives

•	To analyze employee attrition rates

•	To identify factors affecting employee turnover

•	To study attrition by age, gender, education, and job role

•	To evaluate job satisfaction levels

•	To support HR decision-making with data insight 
________________________________________
Tools & Technologies Used

•	Microsoft Excel – Data analysis and dashboard creation

•	Pivot Tables & Charts

•	Data Cleaning & Formatting

•	Basic Excel formulas
________________________________________
Dataset Description

The dataset includes employee details such as:

•	Employee ID

•	Age and Gender

•	Department and Job Role

•	Education Level

•	Marital Status

•	Job Satisfaction

•	Attrition Status (Yes/No)

•	Salary and Experience details
________________________________________
Data Pipeline

1.	Data Collection – HR dataset gathered

2.	Data Cleaning – Removed errors and missing values

3.	Data Transformation – Structured data using Excel

4.	Analysis – Created pivot tables and metrics

5.	Visualization – Built dashboard using charts and KPIs

Dashboard Features

•	KPI Cards:

 o	Total Employees

o	Attrition Count

o	Active Employees

o	Attrition Rate

o	Average Age

•	Visualizations:

o	Donut chart (Total Employees by Gender)

o	Bar chart (Education-wise Attrition)

o	Bar chart (Age-wise Attrition)

o	Tree map (Attrition by Job Role)

o	Pie chart (Department-wise Attrition)

o	Horizontal bar chart (Attrition by Marital Status)

o	Gauge chart (Job Satisfaction Score)

o	Drill-down and filtering options for better analysis

•	Filters (Slicers):

o	Gender

o	Education

o	Department
________________________________________
Key Insights

•	Attrition rate is around 17%

•	Higher attrition observed in certain job roles (e.g., Sales Executive)

•	Younger employees show higher attrition rates

•	Job satisfaction is moderate (~2.7 rating)

•	Majority employees are active with balanced gender distribution
